HOW TO USE
──────────────────────────────────────────────────────────────
Simply open index.html in a modern browser (Chrome, Firefox,
Safari, Edge) on any device.

No server required. No build step. Just open and enjoy.

──────────────────────────────────────────────────────────────
ADD YOUR PHOTOS & VIDEOS
──────────────────────────────────────────────────────────────
1. Place photos inside:   assets/photos/
   Supported formats: JPG, PNG, WEBP, GIF
   Name them: 1.jpg, 2.jpg, 3.jpg ... (or photo1.jpg, img1.jpg)

2. Place videos inside:   assets/videos/
   Supported formats: MP4, WEBM
   Name them: 1.mp4, 2.mp4 ... (or video1.mp4, clip1.mp4)

3. Place music inside:    assets/music/
   File must be named:    birthday.mp3

The heart collage is generated automatically. 
The more photos you add, the fuller the heart!

──────────────────────────────────────────────────────────────
CUSTOMISE THE MESSAGE
──────────────────────────────────────────────────────────────
Open script.js and find the CONFIG object near the top.
Edit the "message" array with your own paragraphs.
